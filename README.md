# csci-380-finalproject
Mental Health App
A web application designed to help users manage daily tasks, build sustainable habits, maintain focus, and track both productivity and emotional patterns through features such as mood check-ins, habit tracking, focus sessions, schedule building, and trend analytics. Many existing apps address these needs individually, combine them without cohesion, or gate core functionality behind subscription paywalls. The project aims to unify the strongest of these features into a single, harmonized, accessible platform where all tracked data works and is analyzed together rather than as disconnected parts. The app is not meant to replace licensed clinical support, but to provide an accessible support mechanism for aiding everyday executive function-based challenges felt by many.
